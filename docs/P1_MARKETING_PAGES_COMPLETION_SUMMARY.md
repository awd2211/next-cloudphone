## 🎨 设计特点

### 1. 统一的视觉风格
- **渐变Hero区域**：所有页面使用相同的紫色渐变背景
- **白色卡片浮动**：Hero下方的数据卡片使用负margin营造浮动效果
- **圆角设计**：borderRadius: 12px 统一圆角
- **阴影层次**：boxShadow营造卡片层次感

### 2. 配色方案
- **主色调**：#667eea (紫色) → #764ba2 (深紫色) 渐变
- **辅助色**：
  - 蓝色 #1890ff（强调、CTA）
  - 绿色 #52c41a（成功、正向数据）
  - 橙色 #faad14（警告、特殊标签）
  - 紫色 #722ed1（技术、创新）
- **中性色**：#f0f2f5（背景灰）

### 3. 排版规范
- **标题字号**：
  - H1: 48px（Hero标题）
  - H2: 32px（章节标题）
  - H3: 24px（卡片标题）
  - H4: 18px（小标题）
- **段落字号**：15-16px
- **辅助文字**：12-14px
- **行间距**：Space组件统一管理

### 4. 响应式设计
- **断点设置**：
  - xs: <576px（手机）
  - sm: ≥576px（大手机）
  - md: ≥768px（平板）
  - lg: ≥992px（桌面）
- **布局适配**：
  - 手机：单列布局（Col span={24}）
  - 平板：两列布局（Col span={12}）
  - 桌面：四列布局（Col span={6}）

```typescript
const Product = lazy(() => import('@/pages/Product'));
const Pricing = lazy(() => import('@/pages/Pricing'));
```

