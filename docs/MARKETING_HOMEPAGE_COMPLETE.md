## 📝 代码规范

### 组件命名

- 使用 PascalCase：`PlatformStats`
- 文件名与组件名一致：`PlatformStats.tsx`
- 设置 displayName：`PlatformStats.displayName = 'PlatformStats'`

### 样式规范

- 使用内联样式（inline styles）
- 统一使用 px 单位
- 颜色使用 16进制或 rgba
- 最大宽度：1200px
- 间距：24px, 48px, 80px

### 性能规范

- 所有组件使用 `React.memo`
- 复杂计算使用 `useMemo`
- 事件处理使用 `useCallback`
- 避免不必要的重渲染
